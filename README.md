# Pilot

Autopilot for Claude Code sessions.

Pilot is a local-first plugin that monitors coding-agent sessions, detects degradation early, tracks token and budget burn, and prepares clean handoffs before quality drops.

## Vision
- Detect session degradation before the user feels it
- Stay zero-config by default
- Never break the user session
- Remain local-first with SQLite-backed state

## Current status
Scaffolding in progress.

## Planned core capabilities
- Session health scoring
- 8 degradation signals
- Handoff generation
- Budget and token tracking
- CLI and slash skills

## Principles
- Zero-configuration
- Zero-overhead
- Never break the user
- Local-first
